# HomeHub 

## What is this project?

This project has been on my mind for some time. I have often thought about creating something that could potentially be of real use to either myself or others, but I never committed to those ideas. This time I will!

This project is meant to be a internal tool for at home. I've got a lot of fun ideas that could be useful or just fun to create. Some of these ideas are: a **calendar** where family members can see each others appointments, a **task manager** where you can track outstanding tasks (those stupid dishes) or creating an **inventory of all our electronic devices**

It will evolve based on what I find fun to work on. It may never see production, but never say never. The fun factor is the driving force behind this project!


## Stack

The project will be have a frontend and a backend, separate of each other. The frontend will be made with *Nextjs* and the backend with *Express*. The application will also use *MongoDB* and *Redis* for storage purposes.
